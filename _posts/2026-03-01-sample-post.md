## Overview

Echoes of the Hollow is a 2D narrative exploration game developed as a collaborative student project. Players navigate a mysterious underground world, solving environmental puzzles while uncovering fragmented memories through interactive dialogue systems.

## Role

As the gameplay programmer, I was responsible for designing and implementing core player systems and gameplay interactions. I developed the player controller using Unity’s new Input System, implemented state-based movement and animation handling, and created reusable interaction frameworks used across multiple levels.

A major focus of my work was ensuring systems were modular and designer-friendly. I built configurable ScriptableObject-driven systems that allowed teammates to adjust gameplay parameters without modifying code.

## Key Contributions

- Implemented player movement, interaction, and animation state management

- Designed a modular prompt/dialogue trigger system

- Built random event and popup systems with configurable probabilities

- Optimized sprite atlas usage to reduce draw calls and improve performance

- Collaborated using Git with feature branching and code reviews

## Technical Highlights

- Event-driven architecture to reduce Update() polling

- Separation of gameplay logic from UI systems

- Clean, extensible code structure following SOLID principles

## Outcome

The project was completed on schedule and showcased at a university games exhibition, receiving positive feedback for responsiveness and polish.
